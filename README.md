## 功能介绍（完整见运行截图）
管理员：基本功能包括用户登录、注册与密码管理，提供导航栏访问首页、音乐库、MV、用户设置等功能模块；管理音乐分类、播放列表及音乐资源，支持歌曲、歌手、专辑和排行榜维护，用户界面设计简洁实用。  
用户：可通过导航栏访问流行、热歌、新曲、经典分类音乐，浏览、搜索、播放音乐；创建、编辑自定义播放列表，查看歌曲信息及排行榜；个性化设置提供主题选择及账户管理，支持音乐的收藏、下载与分享。
